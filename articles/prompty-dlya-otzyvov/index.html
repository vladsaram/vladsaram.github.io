Часть базы промптов для нейросетей. Ответы на отзывы — это репутация и SEO площадки. Нейросеть отвечает быстро и ровно, если задать тон и политику.

Промпты для работы с отзывами

1. Ответ на позитивный отзыв

«Ты — представитель бренда [название]. Напиши тёплый ответ на положительный отзыв. Отзыв: [вставить]. Поблагодари конкретно за то, что человек отметил, без шаблона «спасибо за отзыв». Мягко пригласи вернуться/попробовать ещё. 2–3 предложения, по-человечески.»

2. Ответ на негатив (без оправданий)

«Напиши ответ на негативный отзыв для [бизнес]. Отзыв: [вставить]. Структура: признать чувство клиента → без оправданий взять ответственность → предложить решение → увести в личный канал. Тон: спокойный, уважительный. Не спорь, не оправдывайся. До 4 предложений.»

3. Шаблоны под частые ситуации

«Сделай 6 шаблонов ответов для [ниша] под ситуации: долго ждали, не подошёл размер/товар, понравилось, ошибка с нашей стороны, вопрос в отзыве, необоснованный негатив. Каждый — заготовка с пропусками [деталь], которую легко подставить. Единый тёплый тон.»

4. Просьба оставить отзыв

«Напиши 3 варианта сообщения с просьбой оставить отзыв после покупки [товар/услуга]. Коротко, без навязчивости, с лёгким объяснением, почему это важно. Один — для мессенджера, один — для email, один — для карточки/чека.»

5. Превращение отзыва в контент

«Из этого отзыва сделай: 1) короткий пост-кейс для соцсетей, 2) подпись под скриншот, 3) буллет «нам доверяют» для лендинга. Сохрани смысл, убери лишнее, не выдумывай фактов. Отзыв: [вставить].»
