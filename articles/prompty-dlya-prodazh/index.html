Часть базы промптов для нейросетей. Холодные сообщения, скрипты, отработка возражений, КП — то, что ускоряет продажи в переписке и B2B.

Промпты для продаж и переписки

1. Холодное сообщение (не спам)

«Ты — B2B-продавец. Напиши холодное сообщение в [Telegram/email] для [кому: роль, компания]. Мой продукт: [что], какую боль решает: [что]. Структура: персональный крючок (зачем пишу именно им) → одна ценность → мягкий вопрос-зацепка, а не «давайте созвонимся». До 400 символов. Без «уникальное предложение» и воды.»

2. Серия касаний (follow-up)

«Составь цепочку из 4 follow-up сообщений после первого холодного контакта без ответа. Продукт: [что], ЦА: [кто]. Каждое касание даёт новую ценность (кейс, цифра, инсайт), а не «ну что, посмотрели?». Укажи интервалы. Последнее — вежливый «закрывающий» месседж.»

3. Отработка возражений

«Дай ответы на 7 частых возражений по продукту [что]: дорого, нет времени, уже есть решение, надо подумать, пришлите КП, не сейчас, не уверен в результате. Каждый ответ: согласиться с правом возразить → переформулировать → вопрос дальше. Без давления.»

4. Коммерческое предложение

«Напиши структуру КП для [продукт/услуга] и тексты блоков. Клиент: [кто, его задача]. Блоки: суть на одном экране → что получит в результатах → как работаем → пакеты/цена → ближайший шаг. Тон: деловой, конкретный, выгоды вперёд процесса.»

5. Скрипт первого созвона

«Составь скрипт первого продающего созвона на 15 минут по [продукт]. Этапы: установить контакт → выявить боль (5 вопросов) → связать продукт с болью → назвать следующий шаг. Дай формулировки и 3 вопроса для квалификации клиента.»

Системно, а не вручную

Когда переписок много — нужна автоворонка: бот + n8n за 1 день. А тексты КП и лендингов — в промптах для копирайтинга.

Что делает холодное сообщение рабочим

  • Персонализация в первой строке — почему именно им.
  • Одна ценность, а не список услуг.
  • Вопрос-зацепка вместо «давайте созвонимся».
